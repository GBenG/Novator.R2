Установка для порезки рулонных полимерных материалов

Установка для порезки рулонных полимерных материалов: Предназначена для раскроя рулонных полимерных материалов больших размеров.Все приводы установки работают от пневматики.Раскрой исходного материала осуществляется с помощью нагревателя импульсного типа (струна).Протяжка ручная с жёсткой фиксацией материала. Раскрой происходит по заданию оператора.При большой ширине исходного материала необходимое участие двух человек

Установка для резки рулонных полимерных материалов ШОВ 4500 
Назначение
 Установка ШОВ 4500 предназначена для порезки рулонных полимерных материалов - полимерной сетки, пленки -  в производственных помещениях с температурой от +10°С до +35°С при относительной влажности воздуха 75%. 
Установка обеспечивает выполнение следующих операций:
размещение материала на рулонодержателе (вручную)  
подъём режущей струны до касания материала (пневмопривод) 
нагрев струны (автоматически по временному интервалу) 
просечку контуров в последовательных слоях материала путём подъёма инструмента
 
Принцип работы  
Установка представляет собою напольную свободно стоящую конструкцию на сварном каркасе из профильной стали.
Подключение к сети электрического тока 220В, 50Гц.
Материал размещают в рулонодержателе установки вручную, придерживая края материала. 
Нажатием на пневмопедаль приводят в действие подъем инструмента. 
Скорость подъема инструмента регулируется дросселями на пневмоцилиндрах. 
Выдержав несколько секунд, отпускают педаль.
Режим резки подбирают опытным путем, регулируя параметры «подогрев», «нагрев» и «время нагрева», начиная  с минимальных значений.
